2. Testimony of Trust может быть выполнено только игроком 37-го уровня или выше Этот квест начинается со встречи High Priest Hollint из Oren Castle Town.
3. Test of the Searcher может быть проведен игроком 39-го уровня или выше. Этот квест начинается с Master Luther из Hunters Village.
