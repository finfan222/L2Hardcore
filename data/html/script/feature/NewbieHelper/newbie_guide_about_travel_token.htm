Newbie Guide:
Туристический жетон новичка зарезервирован для игроков 20-го уровня или ниже.
Он позволяет игроку переместиться в начальный город другого племени с помощью Newbie Guides в каждом городе.
Будьте осторожны, поскольку во многих случаях вы можете изучать и практиковать определенные навыки и задания, которые уникальны для города каждого племени.
Как только игрок достигает 20-го уровня, он больше не может телепортироваться, используя жетон путешествия новичка.
